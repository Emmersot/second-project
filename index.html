What Coding Language Should You Learn?

Python

Python is a high-level, interpreted, general-purpose programming language. Its design philosophy emphasizes code readability with the use of significant indentation. Python is dynamically-typed and garbage-collected.

Ruby

Ruby is an interpreted, high-level, dynamic, general-purpose, open source programming language which focuses on simplicity and productivity.

You can learn more about Ruby by going to this link Ruby Wiki

Swift

Swift is a general-purpose, multi-paradigm, compiled programming language developed by Apple Inc. and the open-source community.
